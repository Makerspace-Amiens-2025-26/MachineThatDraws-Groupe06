Programmer

La machine fonctionne avec une chaîne de traitement numérique complète : du fichier vectoriel sur le PC jusqu'au tracé physique sur papier.

Chaîne logicielle complète — 6 étapes

1
Création du fichier (PC)
On part d'un fichier vectoriel (.SVG, .DXF) créé sur Inkscape ou tout autre logiciel de dessin vectoriel.
2
Génération du G-code
Le fichier vectoriel est converti en G-code — une suite de coordonnées (X, Y) et d'instructions de mouvement (vitesse, levée/descente du stylo).
3
Transmission PC → ESP32
Le PC communique via le port série (USB). Le logiciel "Sender" envoie les lignes de code une par une vers la carte.
4
Interprétation — Firmware GRBL
L'ESP32 fait tourner le firmware GRBL. GRBL traduit chaque ligne de G-code en impulsions électriques précises. Il calcule pour chaque moteur le nombre de pas nécessaires pour atteindre la coordonnée cible.
5
Distribution du signal
Le PCB distribue les signaux proprement vers les deux drivers A4988. Il gère aussi l'alimentation 12V nécessaire aux moteurs.
6
Action mécanique
Les drivers amplifient le signal vers les moteurs NEMA 17. Les rotations sont transmises via les courroies GT2 pour déplacer le chariot et le stylo sur la feuille.

Pourquoi GRBL et pas tout coder soi-même ?

Tout coder aurait donné plus de liberté mais aurait pris trop de temps. GRBL est une solution open-source éprouvée et bien documentée qui nous a évité de tout reprogrammer.

Commandes G-code supportées

CommandeAction
G0 X__ Y__Déplacement rapide sans tracé (stylo levé)
G1 X__ Y__ F__Déplacement avec tracé (stylo baissé). F = vitesse en mm/min
M3Baisser le stylo
M5Lever le stylo

Firmware — code principal

// Machine That Draw — Groupe 06
// Makerspace Amiens 2025-2026

#include <Stepper.h>
#include <Servo.h>

const int STEPS = 200;
Stepper motorX(STEPS, 8, 9, 10, 11);
Stepper motorY(STEPS, 4, 5, 6, 7);
Servo penServo;

void setup() {
  Serial.begin(115200);
  motorX.setSpeed(60);
  motorY.setSpeed(60);
  penServo.attach(12);
  penServo.write(90); // Stylo levé
}

void loop() {
  if (Serial.available()) {
    String cmd = Serial.readStringUntil('\n');
    parseGCode(cmd);
  }
}

Exemple de G-code généré

; Dessin d'un carré 50×50 mm
G0 X0 Y0         ; Aller à l'origine
M3               ; Baisser le stylo
G1 X50 Y0 F800   ; Ligne horizontale 50 mm
G1 X50 Y50       ; Ligne verticale 50 mm
G1 X0 Y50        ; Ligne horizontale retour
G1 X0 Y0         ; Fermer le carré
M5               ; Lever le stylo
G0 X0 Y0         ; Retour origine
